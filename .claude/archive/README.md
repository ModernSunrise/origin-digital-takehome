# Archive

Completed plans, finished decision logs, and superseded notes move **here**, out of
the active context Claude loads. History is preserved; the token budget is not spent
re-reading work that is already done.

**Rule of thumb:** when a plan in `.claude/plans/` is fully shipped, or a context file
is replaced, move it here rather than deleting it. Nothing in this directory is
auto-loaded — it is read only on explicit request.
